HoverablePopovers

A simple jQuery plugin to make the twitter bootstrap popovers sticky when hovering.

Usage

<a class="popover-trigger">Trigger popover</a>

<script type="text/javascript">
  $('.popover-trigger').hoverablePopover({delay: { show: 500, hide: 500 }}, 500);
</script>

The hoverablePopover method takes 2 arguments:

  1. PopoverOptions: The standard Twitter bootstrap popover options. (trigger is set to 'manual' by the plugin)
  2. Threshold (optional, default 200): The time from leaving the trigger before the popover is closed if it's not hovered.

Notes

  • There's currently only support for triggering by hover.
